# **App Name**: STUD'IN

## Core Features:

- User Authentication: Secure registration and login functionality using JWT.
- Housing Listings: Browse available housing options with filtering by city, type, and price.
- Carpooling Coordination: Coordinate carpooling opportunities based on departure and arrival locations.
- Tutoring Services: Connect students with tutors, filtered by subject and level.
- Event Listings: List public events for students. Filter events by category, city and university.
- User Profile Management: Users can manage their profile information, including personal details and university.
- Database Integration: The backend will interface with a PostgreSQL database.

## Style Guidelines:

- Primary color: Dark indigo (#3F51B5) to convey a sense of trust, reliability, and intellect.
- Background color: Very light indigo (#F0F2FF) to complement the primary color with a calm, professional feel.
- Accent color: A saturated blue (#03A9F4) for highlights and interactive elements, adding a modern and vibrant touch.
- Body and headline font: 'Inter', a sans-serif font, to ensure readability and impart a contemporary feel across the app.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use modern, outline-style icons to represent different sections and actions within the app. Icons should be simple, clear, and consistent in style.
- Implement a clean, grid-based layout to ensure content is well-organized and responsive across different devices. Use white space effectively to avoid clutter and improve the user experience.
- Subtle transition animations when navigating between pages, and when content loads on a page. Use animation to enhance user engagement and provide visual feedback for actions.
